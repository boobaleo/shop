## Название проекта
-  Интернет магазин брендововй одежды рома и глеба

## Используемая технология
- PHP 7.2
- MySQL 5.7
- Laravel
- OpenServer

## Инструкция по установке 
- cd domains
- git clone https://github.com/boobaleo/shop.git
- cd shop
- composer self-update
- composer install
- создать файл .env из exemple.env
- запустить сайт
